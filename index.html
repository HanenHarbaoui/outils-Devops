OUTILS 1 : GIT

Les commandes de base

  • git init: intier le repot en local
  • git status: lister les changements dans le repertoire de travail
  • git add --all: ajouter les modifications a la staging area
  • git commit -m "Premier commit" : sauvgarde de cette version de projet
  • git log:permet consulter toutes les informations de commit
  • git log --oneline:permet de consulter les differents informations de commit (en résumé)
  • git branch dev : creer une nouvelle branche dev
  • git checkout dev : basculer vers la branche dev
  • git checkout -b test : creer et basculer vers la nouvelle branche test
